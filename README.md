# WALLETPROJECT

## Setup Instructions
### Client
The client folder contains a react app using vite. To get started, follow these steps:

Open up a terminal in the /client folder
1. Run npm install to install all the dependencies
2. Run npm run dev to start the application
3. Now you should be able to visit the app at http://localhost:5173/

### Server
The server folder contains a Node.js server using express. To run the server, follow these steps:

1. Open a terminal within the /server folder
2. Run npm install to install all the dependencies
3. Run node index to start the server
Hint - > Run npm i -g nodemon and then run nodemon index instead of node index to automatically restart the server on any changes!

The application should connect to the default server port (3042) automatically!
